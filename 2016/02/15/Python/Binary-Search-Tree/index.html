Binary Search Tree

Definition

Binary search tree is a node-based binary tree data structure which has following properties:

  • The left subtree of a node contains only nodes with keys less than the node’s key
  • The right subtree of a node contains only nodes with keys greater than the node’s key
  • The left and right subtree each must also be a binary search tree

Advantage of Binary Search Tree

  • Search, insert, and delete opreations are O(log(n)).
  • Get all keys in sorted order by just doing traversal of BST.
  • Doing order statistics, finding closest lower and greater elements, doing range queries are easy to do with BSTs.
